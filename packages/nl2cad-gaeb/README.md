# nl2cad-gaeb

GAEB X81–X85 generator and IFC-to-GAEB converter for the nl2cad ecosystem.

## Installation

```bash
pip install nl2cad-gaeb
```

## Packages

- `nl2cad.gaeb.generator` — GAEBGenerator (XML + Excel)
- `nl2cad.gaeb.converter` — IFCX83Converter
- `nl2cad.gaeb.models` — Position, LosGruppe, Leistungsverzeichnis
